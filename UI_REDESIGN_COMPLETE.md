## 技术亮点

### 1. CSS变量系统
所有设计token统一管理，易于维护和主题切换：
```css
--primary-color: #0ea5e9;
--bg-main: #0a0e1a;
--shadow-md: 0 4px 6px -1px rgba(0, 0, 0, 0.4);
--transition-fast: 0.15s cubic-bezier(0.4, 0, 0.2, 1);
```

### 2. 玻璃态效果
使用backdrop-filter实现真实的玻璃态：
```css
background: rgba(21, 27, 46, 0.85);
backdrop-filter: blur(16px);
```

### 3. 渐变设计
主色和辅助色的渐变，增加视觉吸引力：
```css
background: linear-gradient(135deg, var(--primary-color), var(--accent-color));
```

### 4. 微交互动画
- 按钮悬停：translateY(-1px) + 阴影增强
- 图标缩放：scale(1.1)
- 卡片滑入：slideInRight
- 指示器展开：scaleX(0) → scaleX(1)

### 5. 性能优化
- 使用transform和opacity实现动画（GPU加速）
- cubic-bezier缓动函数，更自然的动画曲线
- will-change提示浏览器优化
