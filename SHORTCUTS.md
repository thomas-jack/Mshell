# MShell 快捷键说明

## 全局快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+N` | 新建连接 | 打开会话配置对话框 |
| `Ctrl+K` | 快速连接 | 打开快速连接对话框 |
| `Ctrl+F` | 搜索 | 在终端中搜索文本 |
| `Ctrl+W` | 关闭标签 | 关闭当前活动的终端标签 |
| `Ctrl+Tab` | 下一个标签 | 切换到下一个终端标签 |
| `Ctrl+Shift+Tab` | 上一个标签 | 切换到上一个终端标签 |
| `Ctrl+,` | 设置 | 打开应用设置面板 |

## 终端快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Shift+C` | 复制 | 复制选中的文本到剪贴板 |
| `Ctrl+Shift+V` | 粘贴 | 从剪贴板粘贴文本到终端 |
| `Ctrl+Shift+A` | 全选 | 选中终端中的所有文本 |
| `Ctrl+L` | 清屏 | 清空终端屏幕（通过右键菜单） |

## 右键菜单

在终端中右键点击可以显示上下文菜单：

- **复制** - 复制选中的文本（仅在有选中文本时显示）
- **粘贴** - 粘贴剪贴板内容到终端
- **全选** - 选中终端中的所有文本
- **清屏** - 清空终端屏幕

## 终端搜索快捷键

在搜索框中：
- `Enter` - 查找下一个匹配项
- `Shift+Enter` - 查找上一个匹配项
- `Esc` - 关闭搜索框

## 使用技巧

### 复制粘贴
1. **方法一**：选中文本后按 `Ctrl+Shift+C`，然后按 `Ctrl+Shift+V` 粘贴
2. **方法二**：选中文本后右键点击自动复制，在空白处右键点击自动粘贴
3. **方法三**：使用右键菜单选择"复制"或"粘贴"

### 快速连接
1. 按 `Ctrl+K` 打开快速连接对话框
2. 输入主机、端口、用户名和密码
3. 按 `Enter` 或点击"连接"按钮

### 标签管理
- 使用 `Ctrl+Tab` 和 `Ctrl+Shift+Tab` 在多个终端标签之间快速切换
- 使用 `Ctrl+W` 关闭不需要的标签

### 终端搜索
1. 按 `Ctrl+F` 打开搜索框
2. 输入搜索内容
3. 使用正则表达式选项进行高级搜索
4. 使用区分大小写选项进行精确匹配

## 注意事项

- 终端快捷键使用 `Ctrl+Shift` 组合，避免与 Linux/Unix 系统快捷键冲突
- 在 Windows 系统中，`Ctrl+C` 在终端中用于发送中断信号（SIGINT），因此复制使用 `Ctrl+Shift+C`
- 右键菜单提供了更直观的操作方式，适合不熟悉快捷键的用户

## 自定义快捷键

目前版本暂不支持自定义快捷键，该功能计划在未来版本中添加。
